## ✅ IntelliJ IDEA 中运行

### 步骤：

1. 打开 `MainWindow.java`
2. 右键点击代码中的 `main` 方法 → `Run 'MainWindow.main()'`
3. 或点击右上角绿色三角运行按钮

> 如果你第一次运行，可能需要配置运行配置（Run Configurations）并指定类路径。
